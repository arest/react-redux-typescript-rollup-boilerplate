# React + Redux + Typescript Boilerplate bundled with Rollup


### Setup:

    npm install

    npm install -g rollup

    npm install -g typings

    npm install -g local-web-server [optional]


    typings install


### Bundle:

trigger rollup manually:

    rollup -c config/rollup.js
        or
    npm run build

run server:
    npm run server
